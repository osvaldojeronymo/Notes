# Estudo das Normas - GELOS

Ao ler as normas internas da CAIXA o funcionário deve-se perguntar:

- quais processos a área efetivamente governa;
- quais são as responsabilidades formais da unidade;
- quais temas são considerados críticos;
- onde estão as interfaces com outras áreas;
- e quais sistemas corporativos aparecem com mais frequência.

Assim, ao final da leitura ele poderá montar uma planilha de análise contendo:

| Campo                   | Descrição                 |
| ----------------------- | ------------------------- |
| Código da norma         | Ex.: NOR 123              |
| Título                  | Nome da norma             |
| Unidade                 | gestora GELOS             |
| Processo principal      | O que a norma regula      |
| Objetivo                | Qual problema ela resolve |
| Sistemas citados        | SIPGC, SAP, SIGA, etc.    |
| Áreas envolvidas        | GELOS, GEINF, SUCOT, etc. |
| Fornecedores envolvidos | Sim/Não                   |
| Contratações envolvidas | Sim/Não                   |
| Patrimônio envolvido    | Sim/Não                   |
| Logística envolvida     | Sim/Não                   |
| Risco operacional       | Baixo/Médio/Alto          |
| Frequência de uso       | Alta/Média/Baixa          |
| Observações             | Insights                  |

Vamos analisar o caso da GN Logística e Suprimentos (GELOS), econtramos em junho de 2026 a seguinte estrutura:

| Tipo                     | Quantidade |
| ------------------------ | ---------- |
| OR - Organização         | 2          |
| MO - Catálogo de Modelos | 19         |
| AD - Administração       | 21         |
| Total                    | 42         |

Isso sugere que a GELOS é uma área predominantemente:

- operacional;
- administrativa;
- orientada a processos padronizados;
- com forte dependência de modelos e fluxos institucionais.

O fato de existirem apenas 2 normas OR indica que a estrutura organizacional da área é relativamente simples. Já a existência de 19 MO chama bastante atenção.

O volume de normas MO indica que boa parte do trabalho provavelmente acontece por meio de:

- formulários;
- modelos;
- catálogos;
- padrões operacionais;
- procedimentos repetitivos.

Isso é típico de áreas de:

- logística;
- suprimentos;
- patrimônio;
- almoxarifado;
- distribuição.
