## 1. Objetivo

Estruturar cooperação técnica entre CAIXA e Arquivo Nacional para digitalização, preservação e disponibilização de documentos do Arquivo Histórico, iniciando pelas cadernetas de poupança.

## 2. Contexto

O Ministério Público Federal (MPF) tem demandado acesso a documentos históricos da CAIXA de interesse público e social. Diante desta solicitação do MPF foi discutido modelo de cooperação com o Arquivo Nacional.

## 3. Principais deliberações

| Tema                 | Definição                          |
| -------------------- | ---------------------------------- |
| Instrumento jurídico | Acordo de Cooperação Técnica — ACT |
| Plano de execução    | Plano de Trabalho vinculado ao ACT |
| Prioridade           | Cadernetas de poupança             |
| Segunda etapa        | Livros históricos                  |
| Custos               | Sem cobrança do Arquivo Nacional   |
| Transporte           | Responsabilidade da CAIXA          |
| Local                | Arquivo Nacional — Brasília        |

## 4. Modelo operacional

```mermaid
flowchart LR
	A[Acervo Histórico da CAIXA] --> B[Avaliação física]
	B --> C[Higienização]
	C --> D[Digitalização]
	D --> E[Preservação digital]
	E --> F[Disponibilização pública]
```

## 5. Responsabilidades

| CAIXA                       | Arquivo Nacional         |
| --------------------------- | ------------------------ |
| Disponibilizar documentação | Enviar modelo de ACT     |
| Transportar acervo          | Avaliar estado físico    |
| Indicar pontos focais       | Higienizar documentos    |
| Apoiar logística            | Digitalizar acervo       |
| Preservar e disponibilizar  | Elaborar ACT em conjunto |

## 6. Próximos passos

| Ação                          | Responsável              |
| ----------------------------- | ------------------------ |
| Enviar modelo padrão de ACT   | Arquivo Nacional         |
| Agendar visita técnica        | Arquivo Nacional         |
| Dimensionar livros históricos | CAIXA                    |
| Indicar pontos focais         | CAIXA                    |
| Elaborar plano de trabalho    | CAIXA + Arquivo Nacional |

## Mensagem-chave

A reunião estabeleceu as bases técnicas e jurídicas para uma cooperação entre CAIXA e Arquivo Nacional, com início pela digitalização das cadernetas de poupança. O próximo marco é a formalização do ACT e a visita técnica para dimensionamento dos livros históricos.
