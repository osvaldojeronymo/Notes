# Metas

Agrupa produtos, estudos e artefatos vinculados às metas dos projetos.

Estrutura:

- Meta 02 - Diagnóstico
- Meta 06 - Políticas
- Meta 07 - Integração
- Meta 10 - Sustentabilidade

Cada meta deve possuir:

- descrição;
- produtos;
- entregas;
- status;
- dependências;
- riscos;
- documentos relacionados.
